# Roman Number Converter

A small and simply roman number converter, Project created with create-react-app

  Demo on the github: https://quinn-h.github.io/roman-number-converter/
### Teck

  - create-react-app
  - Jest
  - clipboard.js

### Install

  To start

```sh
  $ npm Install
  $ npm start
```

  For testing

```sh
  $ npm test
```
